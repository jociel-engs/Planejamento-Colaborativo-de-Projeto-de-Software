# EasySênior

##  Descrição do Projeto
O EasySênior é um sistema desenvolvido para facilitar o agendamento de consultas médicas para idosos, oferecendo uma interface simples e acessível para os usuários que são idosos.

O objetivo principal é reduzir a dificuldade que pessoas idosas enfrentam ao utilizar tecnologias modernas, garantindo autonomia e praticidade no acesso à saúde. 

---

##  Objetivos
- Facilitar o agendamento de consultas
- Melhorar a acessibilidade digital para idosos
- Reduzir erros e dificuldades no uso de aplicativos
- Criar uma navegação simples e clara
